# Tic Tac Toe Game

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**. Play against a friend in this fun browser-based game!

## Features
- Interactive 3x3 grid
- Player X and Player O take turns
- Reset button to start a new game
